Welcome To Connor's Website

This is a CV website I have created as part of coursework in first year as a digital media student. This website was designed to be accessable to all users with a specific focus of functionality. I decided to keep my website fairly basic with large fonts and pictures that are easy to view. I have also made sure the website works well on phones and tablets and the is no functionality errors. I have used ALT text on images and ARIA to help screen readers understand the page. Along with these tecniques i have made it easier for myself in the future to understand the page by keeping the html clean and using Internal Commantary so in future should i need to change the website it will be easy to do.
